# Vanta Domain Context

## Terms

- **Search URL**: The canonical `/search` URL state for query, media type, page, year, sort, and future genre filters.
- **Provider availability**: TMDB watch provider data grouped for a region into Stream, Rent, and Buy rows.
- **Gallery**: A render-ready set of TMDB posters, backdrops, or episode stills shown on entity pages.
- **Recommendation set**: A compact, deduped set of related movie or TV titles merged from TMDB `similar` and `recommendations`.
- **Entity page data**: Server-prepared data for a movie, TV show, season, episode, or person page, including canonical URL metadata.
